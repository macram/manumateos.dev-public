Insertar notas al pie en Markdown

Mar 6, 2022

Escribiendo un artículo para otro sitio me he visto en la necesidad de insertar notas al pie. Como siempre, estaba escribiendo usando Markdown (habitualmente uso iA Writer), así que busqué si había una sintaxis para ello.

Resulta que es algo que en el Markdown original no está definido. Pero sí que existe una sintaxis extendida de Markdown para añadir funcionalidades que no están en el lenguaje original.

La de las notas al pie1 es sencilla. Hay varias opciones.

Texto[^ Nota al pie]

Texto [^1]

[^1]: Nota al pie de varios párrafos.

	Este es el segundo párrafo.

Honestamente no estoy seguro de que el conversor Markdown → HTML que usa Hugo admita las notas al pie; este post es una buena forma de averiguarlo.

  1. Footnotes. ↩︎
